this is for other olds version for open source gcc8 ..and more.

for that install in the last version from 14/13 can you use the commands.

1. cd /
2. wget https://github.com/mambombo777/gcc8-gcc48-gcc49/blob/main/gcc8.tar.gz
                            or
fetch https://github.com/mambombo777/gcc8-gcc48-gcc49/blob/main/gcc8.tar.gz

3. cd /usr/ports/lang/gcc8
4. sudo make ALLOW_UNSUPPORTED_SYSTEM=yes install clean
